# Interview Practice Drills

## 1) API Automation Drill
- Add negative tests for boundary bet values in spins
- Add tests for malformed Authorization headers
- Add race-condition simulation around reward claim requests

## 2) Regression Strategy Drill
- Tag smoke scenarios and run them on every PR
- Run full regression nightly
- Add flakiness dashboard and quarantine process

## 3) Data Integrity Drill
- Add persistence layer (PostgreSQL or MongoDB)
- Validate transactional safety for coin and inventory updates
- Add rollback verification tests

## 4) CI/CD Drill
- Add test report artifact upload
- Add quality gate for coverage threshold
- Add deploy blocking when critical suite fails

## 5) Cross-team Communication Drill
- Write one-page quality strategy for Product + Backend
- Include scope, risks, gates, and defect SLA
